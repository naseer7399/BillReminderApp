# BillReminderApp
# 📱 Bill Reminder App

A simple Android app to track bills, get reminders, and manage due dates.

---

## ✨ Features

- Add, edit, delete bills
- Swipe to delete
- Sort by upcoming bills
- Monthly & yearly repeat
- Notifications (WorkManager)
- Home screen widget

---

## 🛠 Tech Stack

- Kotlin
- Room Database
- WorkManager
- RecyclerView
- Material Design

---

## 🚀 Getting Started

1. Clone the repo:
   git clone https://github.com/YOUR_USERNAME/BillReminderApp.git

2. Open in Android Studio

3. Click Run ▶️
